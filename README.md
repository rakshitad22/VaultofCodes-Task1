# 🚀 Task 1 – Internship Webpage Redesign

This repository contains my submission for **Task 1** of **Assignment 3 – Design & Code Challenge** under the **VaultofCodes Web Development Internship Program**.

The objective of this task was to redesign the provided Internship Programs webpage using only **HTML5** and **CSS3**, while creating a modern, responsive, and visually appealing user interface.

---

## ✨ Features

- Premium Light Theme
- Responsive Design
- Modern Internship Cards
- Interactive "View Details" Button
- Interactive "Apply Now" Button
- Internship Badges
- Hover Animations
- Professional Typography
- Clean UI/UX
- Responsive Layout for Desktop, Tablet & Mobile

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome
- Visual Studio Code
